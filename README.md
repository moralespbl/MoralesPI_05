# Modelo de Scoring Crediticio - Dirty 1
Este proyecto desarrolla un modelo de clasificación binaria para predecir la probabilidad de mora técnica.

### 🚀 Características principales:
* **Dataset:** 26 variables originales reducidas a 13 predictores significativos.
* **Preprocesamiento:** Limpieza de formatos regionales, tratamiento de outliers e ingeniería de atributos (Bins, Logaritmos).
* **Algoritmo:** Regresión Logística con balanceo de clases (`class_weight='balanced'`).
* **Métricas:** Gini > 0.40 y optimización de Cut-off para gestión de riesgo.

### 🛠️ Tecnologías utilizadas:
Python (Pandas, Scikit-learn, Matplotlib, Seaborn).
